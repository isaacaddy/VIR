Usage Instructions:

1. Include the PDF Generator:

<script src="js/dvla-pdf-generator.js"></script>

2. Generate PDF:

generateVehicleRecordPDF(recordId)

3. Print Document:

printVehicleRecord(recordId)

4. Custom Options:

generateVehicleRecordPDF(recordId, { filename: 'custom.pdf' })
